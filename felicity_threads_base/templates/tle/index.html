Instructions

Do not make the mistake of considering this as just another coding contest. Each question here is an exotic research problem in itself. So gear up all your statistics, machine learning, data mining and most importantly your intuition.
For any query, doubts, suggessions or discrepencies contact via
  • Email: kingsofml@felicity.iiit.ac.in

Rules  â–¼
  1. The portal link will be available 5 minutes before the contest.
  2. There are multiple questions, all of which are independent of one another and will have different winnners.
  3. For each problem you will be given a dataset to work on and will be judged on a corresponding test data.
  4. Participants will have to submit a .txt/.csv file along with their code as their submission for every problem they participate in.
  5. If your code contains more than one file, then zip them and upload. Also, if you are using any toolkit like WEKA, then submit the model file. Any code which can ensure that you have not done any kind of hand labelling will be accepted and winners will be contacted at the end of the contest.
  6. The score you will see during the contest will only be partial (obtained on 40% of the test data). The test data you are provided with comprises of only 40% of the whole test data.
  7. Your latest code will be manually evaluated and run on the entire dataset at the end of the contest. Hand labeling will be considered as an act of plagarism.
  8. There is no restriction on the language/library you use.
  9. Participant cannot submit a solution for a problem more than 10 times in a day.
  10. This is an individual event. Any type of plagiarism will lead to disqualification.
